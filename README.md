# Simple JSON Server

Start Simple Json Server

- Auth (JWT or Session)
- CORS
- Todos, Users, Boards...

## installation

```bash
yarn install

cp .env.example .env

yarn serve
```

### urls

- `/docs`: Swagger
- `/auth`: auth with jwt
- `/auth/session` : auth with session

- `/todo`, `/user`, `/board` : simple

- `/mp/**` : Morpheus GW 규격의 대응
